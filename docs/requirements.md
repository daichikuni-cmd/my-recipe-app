# 要件: レシピ管理アプリ（最小MVP）

## 概要
- ユーザーは材料とレシピを登録・参照できる。
- 端末内のみで完結（サーバ不要）。Android 12対応。

## 画面一覧
- 材料一覧（IngredientList）
  - 表示項目: 名前、単位、カロリー/単位、最終価格
  - 操作: 追加、編集、削除、検索
- レシピ一覧（RecipeList）
  - 表示: タイトル、作成日、サムネ
- レシピ詳細（RecipeDetail）
  - 表示: 材料（量）、手順（テキスト＋写真）

## 非機能要件
- 起動時間 < 1.5s（Cold startは除く）
- Roomを用いた永続化
- ローカルユニットテスト有り（主要ビジネスロジック）

## 受け入れ基準（例）
- 材料を追加 → アプリ再起動後も表示される（永続化）
- レシピに写真を追加 → レシピ詳細で表示される
